# My-Portfolio
A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, education, and contact information with a clean and professional user interface.

## 🚀 Features
- Responsive design for desktop and mobile
- Hero / Introduction section
- About Me section
- Skills showcase
- Projects portfolio
- Resume / CV section
- Contact form
- Smooth navigation
- Modern UI design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```text
Portfolio-Website/

├── index.html

├── images/
│   ├── profile.jpg
│   ├── project1.jpg
│   └── ...

```

## 📌 How to Run
1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Run with Live Server or open `index.html` in a browser.
